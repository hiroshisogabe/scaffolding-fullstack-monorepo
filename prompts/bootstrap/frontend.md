<role>

<task>
Implement a frontend in React with minimal viable code.
</task>

<requirements>
# Technical

## Folder structure
- Create the frontend in `./frontend`.
- Keep it completely disconnected from `./backend` (no imports, no shared runtime dependency, no coupling in scripts).
- Create only folders and files required by the implemented solution.
- Do not create empty placeholder directories unless they are used by real code.

## Project initialization
- Bootstrap using Vite.
- Use npm as the package manager.
- Use React v19.
- Enforce TypeScript adoption across application code.

## Scaffolding cleanup
- After initialization, remove irrelevant default scaffolding.
- Remove default Vite/React assets and style files not needed for the final UI (for example, Vite logo assets, extra CSS, unused starter files).
- Keep only the minimum files needed to run the app.

## Application behavior and UI
- Build a single-page frontend with no router.
- Render exactly one centered text on screen: `Generated by AI`.
- Use white background and black text color.
- Ensure the text is centered both vertically and horizontally.

## Code standards
- Prefer `.ts` and `.tsx` files; avoid `.js` unless strictly required by tooling.
- Do not use `any` in TypeScript.
- Keep implementation minimal and clean.
</requirements>

<tests>
## Verification
- Validate the app starts successfully with the Vite dev server.
- Validate the rendered page visually matches the required UI (single centered text, white background, black text).
</tests>

- A short run report containing:

  - command used to run the frontend
  - local URL used for manual verification
</deliverables>
